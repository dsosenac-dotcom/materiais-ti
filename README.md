# 📚 Materiais de TI — Dario Oliveira

Repositório com materiais educacionais produzidos para aulas de Informática,
Robótica, Excel, Linux e Desenvolvimento Web.

---

## 📁 Estrutura

```
materiais-ti/
│
├── 📊 excel/
│   ├── basico/          → Atividades e apostilas para iniciantes
│   ├── intermediario/   → Fórmulas, funções e formatação condicional
│   ├── avancado/        → SOMASES, PROCH, tabelas dinâmicas, Power Query
│   └── dashboards/      → Painéis e relatórios visuais
│
├── 🤖 robotica/
│   ├── spike-prime/     → Projetos com LEGO SPIKE Prime
│   ├── spike-essential/ → Projetos com LEGO SPIKE Essential
│   └── micropython/     → Código MicroPython comentado
│
├── 💻 informatica/
│   ├── slides/          → Apresentações para sala de aula
│   ├── atividades/      → Exercícios práticos
│   └── internet-segura/ → Quizzes e materiais de segurança digital
│
├── 🐧 linux/
│   ├── scripts/         → Scripts Bash úteis
│   └── guias/           → Tutoriais e guias práticos
│
└── 🎮 jogos-educativos/ → Mini-games HTML para sala de aula
```
